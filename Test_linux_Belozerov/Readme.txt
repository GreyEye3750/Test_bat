1) Окружение - "Windows Subsystem for Linux", установленное на Windows 10 приложение Ubuntu 20.04.3 LTS из Microsoft Store;
2) В запущенном Ubuntu установить текущей директорию с папкой, где расположены мои файлы;
   (команда /mnt/c/.../Test_linux_Belozerov, в данном примере у меня папка расположена на диске с, нужно подставить свой путь к папке;
3) В файле Input.txt на второй строке указано время работы программы.
4) Запуск - в терминале набрать ./Start_program.sh
5) Принцип работы:
   	Запускаются ping.exe и ipconfig.exe, первая запускается без времени остановки;
	Каждая из этих программ записывает данные в свой файл.txt;
	Запускается скрипт Cheking.sh:
		Проверяется наличие файла Config.txt, если его нет, то он создается;
		Из него считывается время работы скрипта Т;
		Периодически во временный тхт-файл записываются запущеные процессы;
		До истечения времени работы Т периодически выводится информация о работе программ в консоль и в лог файл с текущей датой.
	По истечению времени Т запускается скрипт Output.sh:
		Во временном файле проверяется наличие искомых программ;
		Если программы все еще запущены, то они закрываются с соответствующим сообщением в консоль и в лог файл;
		Все данные из временных файлов записываются в лог файл.
	Все временные тхт файлы удаляются, остается только тхт лог файл с текущей датой в названии;
	Дата в формате ГГ-ММ-ДД;
	В консоли предлагается повторить скрипт.
